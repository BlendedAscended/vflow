# Verbaflow UI Kit — Agency surface (Studio White)

The `/agency` route. Palantir/Civic × Studio White. Light only — never inherits the dark teal.

## What's in here

| File | What it does |
|---|---|
| `index.html` | The agency page in one HTML file |
| `AgencyNav.jsx` | Transparent nav, ink links, ink primary CTA |
| `AgencyHero.jsx` | Iso-floor video hero + civic-blue "Schedule a meeting" CTA |
| `SocialProofBar.jsx` | Ticker of healthcare / fintech / nursing / trade logos |
| `AgentZones.jsx` | Three zone cards — Healthcare (green), Finance (blue), Cloud (violet) |
| `MetricsRow.jsx` | Chart strip with domain-tinted bar charts |
| `AgencyFooter.jsx` | Quiet ink-on-paper footer |

## Open

```
open ui_kits/agency/index.html
```

## Notes

- `bg #FAFAF8` (warm off-white) — never pure `#FFFFFF`.
- Three domain accents are used **only** in badges, dots, chart strokes — never as fills.
- The `Schedule a meeting` CTA is the only place `#1A73E8` shows up.
